## 项目背景

建立公司记忆沉淀体系，实现知识持续积累和传承。

---

## 项目目标

1. **四维记忆模型落地**
   - L1 公司级记忆
   - L2 VP 级记忆
   - L2 项目级记忆
   - L2 领域级记忆
   - L3 会话级记忆

2. **Git 自动同步**
   - 每小时压缩同步
   - 每天全量同步
   - 会话变更时同步

3. **Token 控制优化**
   - 分级加载策略
   - 自动压缩算法
   - 使用监控告警
